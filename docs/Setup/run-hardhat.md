### Hardhat
  Shenanigan's hardhat package hosts the Shenanigan smart contracts. Hardhat is a compile/testing environment for our solidity code. 
  
  From the shenanigan-monorepo root:
  
  1. install dependencies
       `yarn install`
  2. Run 
       `yarn deploy`
